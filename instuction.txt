**Инструкция по работе с GIT :**

git --version  -*выводим версию GIT*.      
git init   -*создаем репозиторий в новой папке*.
git status   -*проверяем статус происходящего в впапке*.
Создаем файл для работы.
Заполняем файл содержимым.
Сохраняем.
git add -m"название рабочего файла"  -*контроль отслеживания изменений*
